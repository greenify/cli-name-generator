cli-name-generator
==================

generates random usernames

# Dependencies

* you will need ```xsel``` for the interactive mode
* you will need some Perl modules 
* currently unix runs under unix system

# Modes

| Mode | Example    |
| ---  |  ---       |
| 0    | First Last |
| 1    | First.Last |
| 2    | first.last |
| 3    | First-Last |
| 4    | first-last |
| 5    | FirstLast  |
| 6    | firstlast  |
| 7    | Word       |
| 8    | word       |
| 9    | Word.Word  |
| 10   | word.word  |
